# Hubvisor build helpers

Pins common build configuration and dependencies for hubvisor projects.

## Installation

`yarn add @hubvisor-build/core -D`

## Usage

- In your .babelrc:
`"extends": "@hubvisor/build-core/babelrc.json"`

- In your .eslintrc.js
`"extends": "@hubvisor/hubvisor"`
